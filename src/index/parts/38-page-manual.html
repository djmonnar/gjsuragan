사용 설명서 처음 쓰는 분도 이 순서대로 읽으면 전부 할 수 있습니다
시스템 소개 화면 구성 대시보드 배송 관리 배송지도 주간·월간 고객 관리 택배 전달 일괄 등록 자동 처리 배지 사전 자주 묻는 질문

소개이 시스템은 무엇을 하나요?

궁중수라간 반찬 정기배송·선택주문 고객을 한곳에서 관리하는 프로그램입니다. 고객을 등록해 두면 배송 요일에 맞춰 매일 배송 목록이 자동으로 만들어지고, 배송 완료 처리를 하면 잔여 횟수가 자동으로 차감됩니다. 모든 데이터는 실시간으로 저장되어 컴퓨터·휴대폰 어디서 열어도 같은 내용이 보입니다.

하루 업무 흐름 (이것만 기억하세요)

1
아침: 시스템을 열면 대시보드에 오늘 배송할 고객이 자동으로 떠 있습니다. (재개 예정·일정 변경 예약도 이때 자동 반영됩니다)
2
포장·출고: 배송 관리 탭에서 직배송/택배 목록을 인쇄하거나, 택배는 로젠으로 전송합니다.
3
배송 후: 완료된 고객을 체크하고 전체 완료 또는 개별 완료를 누릅니다. 잔여 횟수가 1회씩 줄어듭니다.
4
새 주문이 오면: 우측 상단 ＋ 신규 등록, 또는 일괄 등록 탭에 주문 내역을 붙여넣습니다.

화면화면 구성

  • 상단 헤더: 오늘 날짜, 서버 연결 상태(초록 점 = 정상), 아임웹 취소삭제 알림, 다크 테마 전환, 로그아웃, ＋ 신규 등록 버튼이 있습니다.
  • 탭 메뉴: 대시보드 / 배송 관리 / 배송지도 / 주간·월간 / 고객 관리 / 택배 전달 / 일괄 등록 / 매뉴얼. 배송 관리 옆 숫자는 오늘 배송 건수입니다.
  • 로그인: 직원 전용 이메일·비밀번호로 로그인합니다. 5회 이상 틀리면 잠시 후 다시 시도하세요.

DB대시보드 — 오늘 상황 한눈에 보기

시스템을 열면 처음 보이는 화면입니다. 날짜를 바꾸면 그날 기준으로 다시 계산됩니다.

숫자 카드 6개

  • 전체 고객: 등록된 모든 고객(종료 포함) 수
  • 활성 구독: 지금 정기배송이 진행 중인 고객 수
  • 오늘 배송 / 이번 주 / 내일 배송: 배송 예정 건수
  • 일시정지: 배송을 잠시 멈춘 고객 수. 카드를 클릭하면 고객 관리 탭이 정지 고객만 필터된 상태로 열립니다.

아임웹 취소삭제 알림

아임웹에서 주문이 취소·삭제되면 대시보드 상단에 빨간 알림 박스가 나타납니다. 내용을 확인하고 해당 고객을 정리한 뒤 확인 완료를 누르면 사라집니다.

배송 목록

왼쪽은 선택한 날짜의 배송(🚗 직배송 / 택배 구분), 오른쪽은 내일 배송 예정입니다. 고객명을 클릭하면 바로 수정 창이 열립니다. 아래에는 A·B·C 세트별 활성 고객 수가 표시됩니다.

DV배송 관리 — 매일 배송 처리하는 곳

날짜를 선택하면 그날 배송해야 할 고객이 직배송(사천·진주 등 직접 배달)과 택배(로젠)로 나뉘어 표시됩니다. 어제·내일 날짜로 이동해 미리 보거나 지난 기록을 확인할 수 있습니다.

직배송 섹션

  • A·B·C세트/단품 필터로 원하는 것만 볼 수 있습니다.
  • 🖨 인쇄: 배달용 명단(이름·연락처·주소·현관번호·요청사항)을 인쇄합니다.
  • 배달을 마친 고객은 체크 후 직배송 전체 완료 또는 행마다 완료 버튼을 누릅니다.

택배 섹션 (로젠택배 연동)

1
보낼 고객을 체크하고 선택 로젠 전송 (또는 미전송 전체 전송)을 누르면 로젠에 주문이 접수됩니다.
2
송장번호 조회를 누르면 발급된 송장번호가 표에 채워집니다.
3
발송이 끝나면 체크 후 택배 전체 완료로 완료 처리합니다.
💡 배송 완료를 누르면 잔여 횟수가 1회 차감되고, 잔여가 0이 되면 그 주문은 자동으로 종료 상태가 됩니다. 완료 처리는 고객 관리의 배송 이력에서 언제든 ↩ 취소할 수 있습니다.

MP배송지도 — 직배송 기사님용

  • 오늘의 직배송 고객만 지도와 카드로 보여줍니다. 휴대폰에서 쓰기 좋게 만들어져 있습니다.
  • 왕복 추천 순서: 돌담 출발·복귀 기준으로 효율적인 방문 순서를 자동 계산합니다.
  • 주소 전체 복사: 내비게이션에 붙여넣기 좋게 주소를 복사합니다.
  • 미완료만 보기를 켜면 남은 집만 표시됩니다. 각 카드에서 바로 배송완료 처리할 수 있습니다.

RP주간·월간 — 지난 기록과 앞으로의 일정

주간/월간 달력으로 날짜별 배송 예정·완료 건수를 봅니다. 식자재 준비량을 가늠하거나, 특정 날짜에 몇 집을 배송했는지 확인할 때 사용합니다. 아래 표에는 기간 내 배송 완료 내역이 정리됩니다.

CS고객 관리 — 가장 많이 쓰는 탭

같은 사람(이름+연락처)의 주문은 한 줄로 묶여 보이고, 클릭하면 오른쪽에 상세 정보와 주문별 카드가 열립니다.

찾기

검색창에 이름·연락처·주소·주문번호 아무거나 입력하면 됩니다. 세트/유형(정기·선택)/상태(진행·일시정지·종료)/배송방식(직배송·택배)/잔여 유무 필터와 정렬(최신순·가나다순·잔여순)도 있습니다.

고객 상세에서 할 수 있는 일 (주문 카드의 버튼)

  • 수정: 이름·연락처·주소·현관번호·상품·상태·잔여·배송일정·메모 등 모든 정보 수정
  • 배송완료: 오늘 배송을 완료 처리 (잔여 1회 차감)
  • 택배 복사: 이름·연락처·주소·현관번호를 로젠 형식(탭 구분)으로 복사
  • 일시정지 / 재개: 아래 상세 설명 참고
  • 충전: 구독이 끝나가는 고객에게 잔여 횟수 추가 (충전하면 종료 고객도 자동으로 다시 진행 중이 됩니다)
  • 배송일 변경: 선택주문(1회)의 배송 예정일 변경
  • 삭제: 주문 삭제 — 복구 불가이니 신중히!
  • 완료이력의 날짜 옆 취소: 실수로 누른 배송완료를 되돌립니다 (잔여 복구)

⏸ 배송 일시정지 (여행·휴가 등)

1
고객 상세에서 일시정지를 누릅니다.
2
재개 예정일을 정하면 그날 자동으로 재개됩니다. 비워두면 재개 버튼을 누를 때까지 계속 정지됩니다.
3
정지 중에는 배송 관리·배송지도·택배 전달 목록에서 자동으로 빠지고, 잔여 횟수는 그대로 보존됩니다.
4
재개를 누르면 "배송 요일도 변경하시겠어요?"라고 물어봅니다. 요일을 바꿔달라는 고객이면 확인을 눌러 바로 수정하세요.

📅 배송 요일 변경 — 즉시 또는 예약

고객이 "다음 주부터 요일을 바꿔달라"고 하면 그 자리에서 예약해 두세요. 잊어버릴 일이 없습니다.

1
수정을 열고 새 배송 주기와 배송 일정을 선택합니다.
2
새 일정 적용 시작일에 날짜를 넣으면 → 그날까지는 기존 요일로 배송되고, 그날부터 자동으로 새 요일이 적용됩니다. 비워두면 저장 즉시 바뀝니다.
3
예약된 고객은 목록에 변경예약 배지가 붙고, 상세 카드에서 예약취소도 가능합니다.

기타

  • 아임웹 붙여넣기: 아임웹 주문 내역을 붙여넣어 고객 등록
  • ⬇ CSV 백업: 전체 고객 명단을 엑셀에서 열 수 있는 파일로 저장 — 주 1회 백업 권장

TX택배 전달 — 발송 명단 만들기

1
배송일과 세트/유형 조건을 고르고 목록 생성을 누릅니다.
2
오른쪽 미리보기로 명단을 확인합니다.
3
복사해서 문자·카톡으로 보내거나 ⬇ 저장으로 파일을 내려받습니다.

IN일괄 등록 — 주문을 복사·붙여넣기로 등록

텍스트 붙여넣기 (왼쪽)

스마트스토어 주문 내역을 그대로 붙여넣고 자동 분석을 누르면 이름·연락처·주소·세트·요청사항을 자동 인식합니다. 확인 창에서 내용을 점검·수정하고 등록하세요.

엑셀 업로드 (오른쪽)

스마트스토어 주문 엑셀(.xlsx)을 드래그하면 여러 건을 한 번에 등록합니다.

⚠ 엑셀 업로드 시 세트(A·B·C)를 반드시 확인하세요. 세트가 비어 있는 행은 등록되지 않습니다.

자동시스템이 알아서 해주는 것들

  • 배송 목록 생성: 고객의 배송 요일(조리일)에 맞춰 매일 배송 목록이 자동으로 만들어집니다.
  • 잔여 차감·종료: 배송완료 → 잔여 1회 차감, 잔여 0회 → 자동 종료.
  • 일시정지 자동 재개: 재개 예정일이 되면 자동으로 배송이 재개되고 알림이 뜹니다.
  • 일정 변경 예약 자동 적용: 적용 시작일이 되면 새 배송 요일로 자동 전환되고 알림이 뜹니다.
  • 아임웹 취소 감지: 아임웹에서 취소·삭제된 주문을 대시보드에 알려줍니다.
  • 실시간 동기화: 저장 버튼을 따로 누르지 않아도 모든 변경이 즉시 저장·공유됩니다.

FAQ자주 묻는 질문

실수로 배송완료를 눌렀어요.

고객 관리 → 해당 고객 상세 → 완료이력에서 날짜 옆 취소를 누르면 잔여 횟수까지 원래대로 돌아옵니다.

고객이 "이번 주만 쉬고 싶다"고 해요.

일시정지를 누르고 재개 예정일을 다음 주 배송일로 지정하세요. 그날 자동으로 재개됩니다.

고객이 "다음 주부터 화요일 말고 금요일에 달라"고 해요.

수정에서 새 일정을 고르고 새 일정 적용 시작일에 다음 주 월요일을 넣으세요. 이번 주는 기존대로, 다음 주부터 자동으로 바뀝니다.

구독이 끝났는데 계속 받고 싶다고 해요.

해당 고객의 충전 버튼을 눌러 횟수를 추가하면 종료 상태여도 자동으로 다시 진행됩니다.

이사를 가서 주소가 바뀌었어요.

수정에서 배송지와 현관 비밀번호를 바꾸면 다음 배송부터 바로 적용됩니다.

같은 고객이 또 주문했어요. 새로 등록해야 하나요?

네, 신규 등록으로 추가하면 됩니다. 이름과 연락처가 같으면 고객 관리에서 자동으로 한 사람으로 묶여 "재주문 N건"으로 표시됩니다.

데이터가 안 보이거나 "연결 오류"가 떠요.

인터넷 연결을 확인하고 새로고침(F5) 하세요. 계속되면 로그아웃 후 다시 로그인해 보세요.
