회원 탈퇴
탈퇴 요청 시 고객 페이지 로그인이 차단됩니다. 다만 기존 주문, 배송, 정산 기록은 운영 및 확인을 위해 보관됩니다.
고객앱 사용 매뉴얼
로그인, 주문, 식단 확인, 일정 변경, 정산 확인을 순서대로 따라 하시면 됩니다.
1. 로그인과 자동 로그인
  1. 이메일과 비밀번호를 입력한 뒤 로그인을 누릅니다.
  2. 자동 로그인을 켜면 다음 접속부터 같은 기기에서 로그인 상태를 유지합니다.
  3. 브라우저와 바탕화면에 추가한 앱은 저장 공간이 달라 각각 한 번씩 로그인해야 할 수 있습니다.
2. 주문 탭에서 주문하기
  1. 오전과 점심 전에는 오늘 주문 수량을 확인하고 전송할 수 있습니다.
  2. 오늘 주문이 마감된 뒤 오후 1시 전까지는 다음 주문 오픈 안내가 표시됩니다.
  3. 오후 1시 이후에는 주문 탭이 다음 영업일 주문 화면으로 자동 전환됩니다.
  4. 수량을 바꾸려면 +, − 버튼을 눌러 조정합니다.
  5. 요청사항이 있으면 메모에 적고 주문 전송하기를 누릅니다.
  6. 전송 완료 표시가 보이면 주문이 접수된 상태입니다.
마감 후 오후 1시 전에는 “다음 주문은 오후 1시에 열립니다” 안내가 표시됩니다.
3. 다른 날짜 주문과 자체 휴무
  1. 평일 일정 탭에서 원하는 날짜 카드를 누릅니다.
  2. 해당 날짜의 수량을 수정한 뒤 다시 전송할 수 있습니다.
  3. 그날 식사를 받지 않으면 자체 휴무로 등록을 누릅니다.
4. 식단 확인
  1. 식단 탭에서 날짜를 선택합니다.
  2. 조회 버튼을 누르면 해당 날짜의 도시락/샐러드 메뉴를 볼 수 있습니다.
  3. 등록된 식단이 없으면 업체에 문의해주세요.
5. 내 정보와 기본 수량
  1. 내 정보 탭에서 업체명, 연락처, 식사 받는 시간을 수정합니다.
  2. 매일 같은 수량이면 매일 같은 수량 사용을 체크합니다.
  3. 요일마다 다르면 체크를 풀고 월~금 수량을 각각 입력합니다.
기본 수량은 주문을 따로 보내지 않은 날의 기준 수량으로 사용됩니다.
6. 정산 확인
  1. 정산 탭에서 확인할 월을 선택합니다.
  2. 조회 버튼을 누르면 해당 월 청구 내역과 입금 상태를 확인할 수 있습니다.
  3. 금액이 다르면 주문 수량과 배송 완료 내역을 업체에 문의해주세요.
