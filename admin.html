매일 쓰는 순서대로 정리했어요

주문 확인, 배송 완료, 정산, 행사도시락, 식단 입력까지 헷갈리기 쉬운 작업만 빠르게 볼 수 있습니다.

기본 흐름 주문 확인 → 수량 수정 → 배송 완료
1. 주문 현황
  1. 날짜를 선택하면 해당 날짜의 업체별 주문이 보입니다.
  2. 수량이 실제 배송과 다르면 도시락/샐러드 숫자를 수정합니다.
  3. 배송 나간 업체만 체크한 뒤 배송 완료를 누릅니다.
  4. 배송 완료된 건은 정산에 자동 반영됩니다. 직원 업무는 여기서 끝납니다.
고객이 자체 휴무로 보낸 날은 휴무 표시가 뜨며 배송 완료 대상에서 제외됩니다.
2. 회원 관리
  1. 상단의 회원관리 탭에서 업체 목록을 엽니다.
  2. 업체 카드를 클릭하면 업체명, 배송지, 식사시간, 요일별 기본 수량을 수정할 수 있습니다.
  3. 매일 수량이 같으면 매일 같은 수량 사용을 체크합니다.
  4. 요일마다 다르면 월~금 수량을 각각 입력합니다.
3. 행사도시락
  1. 행사도시락 탭에서 도시락명, 날짜, 시간, 장소, 개수, 단가를 입력합니다.
  2. 저장하면 행사도시락 목록과 정산에 함께 반영됩니다.
  3. 변경이 생기면 목록에서 수정 후 다시 저장합니다.
업체가 별도로 주문한 행사도시락은 정산에서 수정할 수 있게 남겨두면 됩니다.
4. 정산
  1. 월말에 사장님이 정산월을 선택하고 불러오기를 누릅니다.
  2. 주문 탭에서 배송완료한 도시락, 샐러드, 행사 수량과 금액만 확인합니다.
  3. 정산 탭에서는 입금완료, 보류, 조정금액처럼 예외만 수정합니다.
  4. 필요하면 정산표, 카톡 내보내기, 영수증을 사용합니다.
5. 식단 관리
  1. 날짜를 고르고 도시락/샐러드 메뉴를 입력합니다.
  2. 저장하면 고객앱 식단 탭과 오늘 주문 화면에 표시됩니다.
  3. PDF나 사이트에서 가져온 내용은 저장 전 후보 내용을 꼭 확인합니다.
6. 설정과 휴무일
  1. 설정 탭에서 고객 주문 마감 시간을 정합니다.
  2. 회사 자체 휴무일은 날짜와 이름을 추가합니다.
  3. 주말과 공휴일은 기본적으로 배송 없음으로 처리됩니다.
